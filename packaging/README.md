# Letter Me This — Packaging

Print-ready packaging artwork for *Letter Me This!* by Nicholas Street Games.

These are **Design Component** files (`.dc.html`). Each one opens directly in a
browser and renders on its own — they only need the two supporting pieces in
this folder:

- `support.js` — the runtime every `.dc.html` loads (keep it alongside the files)
- `assets/` — shared imagery (logo, hero, QR, seals)

## Files

| File | What it is |
| --- | --- |
| `Letter Me This Tri-Fold.dc.html` | Tri-fold instruction insert — working canvas with fold/bleed/safe guides |
| `Letter Me This Tri-Fold-print.dc.html` | Same tri-fold, print layout (11×8.5″ landscape, guides off, auto-print) |
| `Letter Me This Tube.dc.html` | Cylinder / tube wrap — v1 |
| `Letter Me This Tube v2.dc.html` | Cylinder / tube wrap — v2 |
| `Letter Me This Tube v2 - Print Legal.dc.html` | Tube v2 sized for Legal print |
| `Letter Me This Brand Pack.dc.html` | Brand palette, type, and logo reference |

## Printing

Open any `-print` file in a browser and use **Cmd/Ctrl + P → Save as PDF**.
Page size is preset per file; leave margins at "None" / default.

## Colors (quick reference)

- Teal `#1aa3c2` · Red `#f14a3d` · Yellow `#ffce1f` · Green `#2e7d46` · Blue `#2e6db4`
- Ink `#123a47` · Body `#3b555f`
- Fonts: **Fredoka** (display) + **Nunito** (body), both Google Fonts
